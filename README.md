# ecommerece_prototype

A modern, responsive e-commerce storefront frontend built with React and React Router.

## Features

- Home, products, product detail, cart, checkout, account, auth, and 404 pages
- Responsive layouts for mobile, tablet, and desktop
- Cart and wishlist persistence with localStorage
- Functional filters, sorting, skeleton loaders, toast notifications, and dark mode

## Scripts

- `npm start`
- `npm test`
- `npm run build`
